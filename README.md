# Advanced Disk Scheduling Simulator (Full)

Features:
- Algorithms: FCFS, SSTF, SCAN, C-SCAN, LOOK, C-LOOK
- GUI with linear view and rotating disk animation
- Scheduling timeline chart (Chart.js)
- CSV import/export (PapaParse)
- Export visualization to PNG (html2canvas)
- Dark mode
- Sound for head movement (howler)

Run:
1. npm install
2. npm run dev
3. Open the URL printed by Vite (e.g., http://localhost:5173)
